## Projeto em destaque

### [BestSignal - análise EV+ com FastAPI](https://github.com/IgorAlcan/best-signal)

Projeto de portfólio em Python que simula uma mesa simples de análise de valor esperado (EV+). O foco é mostrar organização técnica, clareza de código e capacidade de transformar dados em uma aplicação revisável.

**O que o projeto mostra:**

- API em camadas com FastAPI e Pydantic
- Dashboard com Streamlit
- Frontend web em tema dark
- Dados simulados para análise de sinais
- Testes automatizados com Pytest
- Docker, CI com GitHub Actions e documentação para recrutadores
